# 问:在教会中姊妹也是祭司，她们那一分如何被成全摆在教会中？姊妹几占三分之二，若他们中间有作先知的一份，有怜悯人的一份，如何去帮助她们尽功用？她们要守住她们的地位，但也要尽祭司职分才行啊。

# 答:这也是属灵的问题，不是道理讲讲可以的。在属灵的工作上，弟兄姊妹自己都会有感觉，他们会知道有哪一份，也会把那一份拿出来。

# 也有的人在生命的基本认识上不清楚，他有哪一份，自己不晓得；去做都凭着自己做，不按原则做，就越做越乱。因此对怎样配搭、事奉，也不是说人要怎样做的问题。　　

# 你说:要给她知道她的地位。这个也是圣灵的工作，凭着的，也是生命的感觉; 罗8:6; 约5:12; 林前2:15; 罗8:14）。不然，她有什么地位，她也不清楚。你信主，她也信主，都一样；她该站的地位，都是因着碰到权柄，碰到那个实际才懂得呢。这样，她才可好好的站在她的地位。不然，恐怕她还要站在别人的地位了。　　

# 我们这些人排起来，自然就站好自己的地位。排对了，前面都是一层层的，非常清楚。要不然，你就觉得有高有矮，像很乱的样子。你是乱排的，排到这一面就发觉别人比你高，排到那一面，就突出来，觉得不好意思，于是，到处跑来跑去，跑得很多。　　

# 所以属灵的东西、身体的感觉、权柄的东西，都有属灵的＂排队＂。

# 就是你们那边长老、负责弟兄、同工有几十位，也是要"排队"。这个排队，并不是人老了，就一定是在前面；有的老了还很胡涂的。这个排队是根据属灵的东西。有了属灵的东西，在工作中，大家就知道谁的份量大。根据属灵的东西，大家都知道，份量有多少，自己要站在哪个地位上；大家就会站牢自己该站的地位。如果你这个不认识，就会有意见，摆在配搭里，意见就出来。配搭的人那么多，这有意见，那有意见，意见就多。如果大家都在属灵的实际里，人多意见多也不怕，因为出于神的东西，意见多，也不会把属灵的东西推倒。如果是出于人的，几个意见来，就撑不住。
