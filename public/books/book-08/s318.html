# 答:我们要开口，开口讲话。虽然说出来的有对的，也有不对的，有属灵的东西，也有些人的东西，越说也越产生混乱，但是无形中，再让你讲，你就会进步。有时越讲，你里面越失望，装在心里面存着矛盾，一旦，你把它讲出来，就好了。　　

# 问:有弟兄姊妹病得厉害，情况实在很不好，像有人中风好长时间，不省人事，可否祷告说:＂主快接他去，让他安息。＂可以这样祷告吗？当然要加一句:让你的旨意成就。

# 答:他是一个人，情况就比较简单；如果家庭的儿女都在，也有人求主留他，你却叫他赶快＂去＂:＂主呀！你接他去。＂到底主要听谁的，要接他去还是把他留下来？　　

# 面对这种问题，不管如何，我们要有一个态度:他是病了，我们就求主给他有在病中的恩典；他要临终，就求主给他有临终的恩典。因为没有恩典，死也很难去死，是苦死了，是不是？所以不信的人会骂人说:＂你不得好死。＂我们以前不懂得这句话；现在就懂得了，不好死很苦呀！起又起不来，苦上加苦，身体痛苦，精神压力也苦。子孙孝顺还好一点，不孝顺的会想:＂你不会早死，在这里就给我们拖累。＂又是增加一个精神的压力。所以他们才需要临终的恩典。这情景下要为他祷告，叫他有恩典;雅4:6; 彼前5:10）去忍受，因为这个恩典就是超越的大能力，能够叫他经过约但死河。

# 我们有一位姊妹说:"约但河这死河是很难过的。死河是很难过的！"所以面对人生都要求主怜悯。活着也求主怜悯，病中也求主怜悯。晚年的，求主怜悯给他晚年的恩典，不然，晚年没有恩典是很难受的。
