# 问:我跟一个人传福音，他表示很好，说:"我要回去考虑考虑。"一考虑，跟他妈妈商量就要退(他们原是拜偶像的)。我们是由他呢，还是继续找他？

# 答:这要看主怎么带领。觉得怎样就怎样，还要讲就讲。另一面，我们不要看表面，只听他说＂我信＂，他是表现得很好，但是心里很刚硬。所以你不要给他外面的态度、假相骗去。真的要他信时，他就不信。有的人嘴巴很硬，要跟你争，跟你辩，但是他心里已经给你摸着了，不过是不好意思说:＂我要信。＂这样的人就不要怕他反对，他越反对越信得快！没有反对而赞成的反而不容易信。

# 传福音，如果遇到人对你说:＂呀！你传福音，传得很好，很对，我很赞成。＂他反而不容易信。如果你传福音给他，他反对，跟你争，跟你辩，这样的人反而更踏实。就如买东西的人，满口说好，这个好，那个好；卖东西的人就知道他不会买。因为讲＂好＂的，就是不会买；那想买的，口里都说这不好，那也不好，目的要减价。那说好的却是没有钱拿出来，卖东西的人就知道他不会买。

# 有一次在杂货摊，我说这东西很漂亮，卖东西的人说:＂朋友，你不会买呀！＂我说:＂奇怪，你怎么知道我不会买？＂他说:＂你称赞好，称赞好的，就不会买，如果你说不好，不对，你是想价钱少一点，你就会买。＂所以不是看外面怎样，乃是看他里面。

# 传福音,20; 林前9:16）也一样，对你说好的，他就不信。有时我们给人家传福音，我们一直观察他的思想、动态，观察他的表情，讲到他整个都给我们拉到主这边来了。知道了他那个情形，我们就好抓，如果你不认识他的变化，从头到尾你摸不清，你抓不准，你讲没有用。但是你认识他，你给他说，谈到那里，思想感情也会跟到那里；如果他认真听，感觉到了，思想、感觉就被带到主那边来了，他跑不了！;林前4:15; 罗1:15-7; 徒26:18)这个就是属灵的实际的学习。身体的交通也是如此)，你跟他谈，谈也是要有耐心，哦！找到一条路了，就把他找来跟随主，他就跑不了。
