十一月十一日

# 问:有些人临终才接受主，他有没有得胜？

# 答:这个要这样看，主观来说，他是仅仅得救，只信主得救，当然还没有得胜的经历。但是，客观的说，基督的生命就是得胜的生命，并不是我们今天来做，做出得胜。得胜不是我们做出来的，乃是基督的生命。基督的生命，就是得胜的生命。 ;林前15:57; 罗8:37; 林前15:54,45)

# 交通一:曾听弟兄讲，他到老时，还能接受主，很高兴，很开心；后来却又不说下去，因他想，这么老才信主，没有多少经历，很空的样子，到见主时，生命很浅，没有丰富的生命，这也是个难理解的情形。

# 交通二:主耶稣曾讲进葡萄园做工的比喻。有人先进园，也有人后进园，但工价都一样！这给我们看见:不是我们做了多少，乃是祂的怜悯和恩典。有的信徒信了好多年，到老，还留在错误的路上，没有走在正路上。信主几十年的也不一定得胜。也许有人年日很短，却蒙神怜悯，单纯的就走上了主的路。

# 交通三:人的一生不简单！

# 问:想问关于信心的问题。我的信心很低，怎样操练信心？

# 答:这信心不是自信。在世界，有自信的人──自己信自己的人，都很成功。但我们的信心不是自信，我们信心的对象是神，所以你对神要认识，对祂才有信心。你的父母，你对他们的信心自然比别人更深，是不是？因为父母为着你，总不会害你。你把钱寄在他们那里，你信他们总不会去偷。你什么都信他们，因为是你的父母。你对他们，自然而然有这样的信心；对别人就没有这个信心。如果是别人，你交给他一百块钱，心会常想: 恐怕会给他偷去。

# 现在，我们的神是我们的父，心里必然很踏实和相信他。事情成全不成全，反而次要，在祂手里，我们就有安息。事情若是成全了，你感谢神，还没有成全，你也感谢神，这个心就平安了。
