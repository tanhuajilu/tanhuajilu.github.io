# 问:年长的弟兄邀我们出去，总要我谈他那个。譬如对别人说属灵的，总要把原文或背景解释很透彻。我觉得不必要。我们很难影响他。

# 答:这就是神学的理论和道理，这样的话也许有它的对象。初传福音，去讲这些，初听的听不懂，这样子多几次同工，你多几次难受；你很难受是因为叫你讲这些，你讲了，里面没有膏油。如果有膏油; 133:2; 传9:8; 歌1:3; 约一2:20,27），是生命的东西，他交通给你，你也会接受，也会交通出去，就越讲越有膏油。有膏油就有果效。　　

# 交通:交通给我看见同工不是叫一个人去作伴。如果彼此不同心，道路不同，认识不同，这样同工就有问题。有时，我也感到对坐在会中的人，如果没有把握好，只碍于人情请他发言，一场很好的聚会也会变差。

# 答:被邀发言的人，平日你们要认识他的情形；情形是不对头的就不要请，因你们在会中讲到神的生命，交通到属灵的，已经有一条线，有一种情形，发言的人不合适，会把原来的这个打乱。所以我们最怕在聚会中有随便讲话的人。对聚会一定要有人负责。有负责的人，就能把握住圣灵工作到什么程度，该停就停，该继续可继续下去。你没有摸到圣灵的水流，任凭讲的人"停也好，不停也好"，讲到天地之外，还不知道。这个是要有管理的。所以一个交通聚会，该有人负责去管，不然就乱了。
