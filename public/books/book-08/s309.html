# 问:教会是建好还是没有建好？抑或到新耶路撒冷才建好？

# 答:教会是一直增加的。说建好也可以，说一直加上去也可以，这是相对的。　

# 从使徒行传开始，教会就在那边，历世历代，虽然通过好多的难处，遭受迫害，教会还存在，因为那个复活的能力，属灵的力量还存在。

# 神不是看外面这么多的难处，神是看属灵实际的那些东西。那些东西是属灵的，就是美好的，连在基督身上就是完整的。今天，属灵的东西一直在增加，越增加就越丰满。像你这个人长得好像已经差不多了，但是，你越吃也越胖，越吃也越丰满，达到成熟。

# 基督的教会就是充满基督而成了丰满的)，因此很难说是不是已经完全了，可以说"完全"，也可以说还没有完全。因为神有一个计划，这个计划一定要完成。不完全，因着相信，也必完全，这个完全必定要得到实现──到了那一天，新郎和新妇在那边相见，就完全了。
